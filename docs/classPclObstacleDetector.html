Detailed Description

ROS2 node for detecting obstacles from point cloud data.

This node subscribes to point cloud data from a depth camera, applies filtering to remove irrelevant points, segments the ground plane using RANSAC, performs Euclidean clustering on the remaining points, and publishes 3D bounding box detections for detected obstacles.
