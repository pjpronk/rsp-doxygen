Detailed Description

ROS2 node for controlling a robot to navigate around obstaclesand stop at a pedestrian.

This node subscribes to 2D pedestrian detections and 3D obstacle detections, then publishes velocity commands to navigate the robot. It stops when pedestrians are too close and steers around detected obstacles.
